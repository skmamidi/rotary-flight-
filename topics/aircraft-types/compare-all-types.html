Quick Definitions

Helicopter

Powered main rotor. Can hover. Most common rotary-wing aircraft.

Autogyro

Unpowered rotor (autorotates). Needs propeller for thrust. Cannot hover.

Tiltrotor

Rotors tilt between vertical and horizontal. VTOL + fast cruise.

Tandem Rotor

Two large rotors, front and rear. Heavy lift. No tail rotor.

Coaxial Rotor

Two rotors stacked, spinning opposite directions. Compact, no tail rotor.

Olympiad Edge

State tests often use elimination questions. Practice asking yourself:

  1. Does it hover? If yes, eliminate autogyro.
  2. Is the lift rotor powered? If no, it's an autogyro.
  3. Does it have a tail rotor? If no, it might be tandem or coaxial.
  4. Do the rotors tilt? If yes, it's a tiltrotor.

Key Vocabulary

Powered rotor
Rotor driven by engine to create lift.
Unpowered rotor
Rotor that spins from airflow, not engine.
Counter-rotating
Rotors spinning in opposite directions to cancel torque.
